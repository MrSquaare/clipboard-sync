# Technology Stack

## Client
- **Framework**: [Tauri](https://tauri.app/) (v2)
- **Backend Language**: [Rust](https://www.rust-lang.org/)
- **Backend Cryptography**: [RustCrypto](https://github.com/RustCrypto)
- **Backend Secrets Storage**: [Keyring](https://github.com/open-source-cooperative/keyring-rs/wiki/Keyring) (v3)
- **Frontend Language**: [TypeScript](https://www.typescriptlang.org/) (v5.9)
- **Frontend Framework**: [React](https://react.dev/) (v19)
- **Frontend State Management**: [Zustand](https://github.com/pmndrs/zustand) (v5)
- **Build Tool**: [Vite](https://vitejs.dev/) (v7)

## Server
- **Framework**: [Hono](https://hono.dev/) (v4)
- **Language**: [TypeScript](https://www.typescriptlang.org/) (v5.9)
- **Build Tool**: [esbuild](https://esbuild.github.io/) (v0.27)

## Schemas
- **Schema Definition**: [Zod](https://zod.dev/) (v3)
- **Build Tool**: [tsdown](https://tsdown.dev/) (v0.20)

## Tools
- **Package Manager**:
  - Node: [pnpm](https://pnpm.io/) (v10)
  - Rust: [cargo](https://doc.rust-lang.org/cargo/)
- **Linting/Formatting**:
  - TypeScript: ESLint, Prettier
  - Rust: clippy, rustfmt
