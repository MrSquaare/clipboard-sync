# Clipboard Sync Specs

Welcome to the technical specifications for the Clipboard Sync project.

## Project Overview
This project is a browser extension that enables peer-to-peer clipboard synchronization between two devices without a central server. It uses Web Real-Time Communication (WebRTC) with a manual "copy-paste" signaling handshake.
