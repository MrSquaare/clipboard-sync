# Technology Stack

This project uses the following key technologies and libraries:

## Core
- **Language**: [TypeScript](https://www.typescriptlang.org/) (v5.9)
- **Framework**: [WXT](https://wxt.dev/) (v0.20) - Next-gen Web Extension Framework
- **Bundler**: [Vite](https://vitejs.dev/) (v7)

## UI Library
- **Library**: [React](https://react.dev/) (v19)
- **Renderer**: React DOM (v19)

## Code Quality & Tools
- **Linter**: [ESLint](https://eslint.org/) (v9)
- **Formatter**: [Prettier](https://prettier.io/) (v3)
- **Package Manager**: pnpm
